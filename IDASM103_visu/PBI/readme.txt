Lors du chargement du fichier, vous rencontrerez une erreur au chargement du dossier.
PBI va chercher les csv sur votre disque dur, hors il ne connait pas le chemin vers les documents à charger (*.csv)

Pour charger les données, importer les documents se trouvant dans le sous dossier /data

Dans PowerBI, aller dans "transformer les données" ou "tranform data".
Vous trouverez dans les queries, un parametre "Path".
Dans "current Value", introduisez le chemin jusqu'à "IDASM103_visu" inclus.

faites "fermer et appliquer" ou "close & apply" .

Les données devraient être chargées.
